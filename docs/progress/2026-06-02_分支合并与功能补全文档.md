## 2. 新增功能

### 2.1 前端外观与画布体验

- 新增 `appearanceStore`, 持久化主题和画布外观偏好。
- 支持 `system` / `light` / `dark` / `contrast` 主题。
- 支持 `grid` / `dots` / `paper` / `blueprint` / `plain` 画布背景。
- 支持网格坐标标签开关。
- 房间页新增画布状态读数: 缩放比例、对象数量、选中数量、当前背景。
- 弹层支持 Escape 统一关闭, 覆盖快捷键、外观设置、诊断抽屉和右键菜单。

### 2.2 协作诊断与可观测性

- 新增 `DiagnosticsDrawer`, 展示连接节点、WebSocket URL、重连次数、pending ops、恢复/回放/flush 统计。
- 前端定时读取 `/api/cluster/cache-stats` 和 `/api/cluster/queue-stats`。
- 顶部诊断条展示缓存命中率、队列长度、transient drop 和 overload disconnect。

### 2.3 成员权限与会议同步

- 登录用户进入授权房间时自动登记为成员。
- owner 可以新增成员、调整角色、移除成员。
- 支持 owner 转移, 同时保证房间至少保留一个 owner。
- 成员变化后广播 `room-members`, 客户端刷新自身权限与成员列表。
- MeetingBar 支持会议阶段、阶段列表、聊天、表情、历史回放、AI、成员权限和语音入口。

### 2.4 分布式基准测试与验证脚本

- 新增 `scripts/distributed-benchmark-report.mjs`, 自动生成分布式基准测试报告。
- 新增 `scripts/run-distributed-benchmark.ps1`, 方便 Windows 运行。
- 新增 `scripts/generate-board-fixture.mjs`, 用于生成大白板 fixture。
- 新增独立校验脚本:
  - `scripts/join-token-check.mjs`
  - `scripts/permission-check.mjs`
  - `scripts/history-anchors-check.mjs`
- 新增 `scripts/start-frontend-dev.ps1`, 配合拆分后的 API / WebSocket Vite 代理。
- 新增 `artifacts/distributed-benchmark/` 下的稳定结果样例和分析报告。

### 2.5 测试覆盖增强

- 新增房间成员和账号权限集成测试。
- 新增 history anchors 集成测试。
- 新增 join token 和写入权限策略测试。
- 新增 AI 输出安全清洗边界测试。
